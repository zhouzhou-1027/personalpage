项目介绍

医患无障碍沟通系统

Medical Communication System

这是一个面向听障人士的医疗问诊辅助平台。系统整合了Web Speech API 语音识别、 手语数字人展示、实时字幕翻译和SQLite 数据持久化等功能， 帮助医生与听障患者实现无障碍沟通。

语音转文字

医生语音实时转录为文字，支持快捷短语

手语数字人

CSS 动画手语人，实时展示翻译状态

数据持久化

SQLite 本地数据库存储，支持历史查询
